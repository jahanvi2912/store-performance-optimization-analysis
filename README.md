# Store Performance Optimization Analysis

## Project Overview
This project analyzes retail store performance using Python and data-driven techniques such as exploratory data analysis, feature engineering, and regression modeling to generate business insights.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
The dataset contains:
- 8,190 records
- 12 features
- Promotional markdown variables
- Economic indicators (CPI, Unemployment)
- Holiday indicators
- Seasonal variables

## Key Analysis Performed
- Data Cleaning & Missing Value Treatment
- Exploratory Data Analysis
- Correlation Analysis
- Feature Engineering
- Linear Regression Modeling
- Business Recommendations

## Key Insights
- Holiday promotions were over 3x higher than non-holiday periods.
- Promotional activity was highly event-driven.
- Fuel Price showed strongest modeled influence.
- Store performance drivers were multifactor in nature.

## Model Performance
- R² Score: 0.092
- RMSE: 1.764

## Business Recommendations
- Optimize holiday promotions
- Use seasonal demand planning
- Incorporate macroeconomic indicators into strategy
